# xx海关检验委托业务管理系统

## 业务需求
1、	委托单登记功能
接样人员在Lims里录入委托单信息。一定要录入联系人手机号。（委托单什么时候生效？）
2、	委托单导出导入
接样人员把lims里委托单信息导出，导入业务管理系统。
系统根据“委托单位”，自动确定是长期客户，还是短期客户，如果长期客户，关联账期和折扣，并自动计算交款截止日期。交款截止日期=送样日期+7个工作日，如果委托单位有合同，根据合同约定的账期确定，交款截止日=结款周期结束后+15个工作日。（委托单位信息在“客户管理”里维护）

3、	委托单上传功能
接样人员把纸质委托单扫描生成pdf文件，选择对应的委托单上传业务管理系统。
4、	委托单审核功能
审核人员根据纸质委托单核对业务管理系统数据进行核对。（为什么不在lims直接核对，修改）
5、	委托单报价功能
报价人员以委托企业为单位，选择一条或多条样品数据，选择折扣生成账单。
6、	账单网上审核功能
根据规则，账单通过网上进行审核。
账单审核规则，马艳艳八折以上（含八折），刘星，张文贵七折以上（含七折），曹锡忠，刘星，7折以下（不含七折），双人同时审批。
7、	账单自动发送功能
客服人员将通过审核的账单发送给客户确认。
系统自动通过短信给客户发送账单链接，账单上有业务章水印，客户可以自己下载账单，如果对账单没有异议，对帐单进行确认，并支付相应的费用。
客服人员可以手工补发或者重发。
8、	开票功能
财务人员根据银行进账单开发票，并对账单进行核销，填写发票号、收费单位、到账日期等信息。并打印账单作为财务凭证。
财务人员从网银系统导出进账单excel文件，并根据进账单开发票，把发票号维护到进账单excel文件里，然后把进账单excel导入系统，系统自动对账单进行核销，并能批量打印核销的账单。
9、	报告登记功能
客服人员收到报告后，扫描成pdf文件上传业务管理系统。
10、	电子报告自动发送功能
明确电子报告自动发送规则：已缴费，或者委托企业是长期客户，并且没有欠费；
系统自动通过短信给客户发送电子报告的链接，客户可以自己下载。
11、	纸质报告邮寄登记功能
客服人员收到发票后，进入系统查询发票对应的报告，把发票和发票对应的报告邮寄给客户，并登记发送时间、发送人员、快递公司、快递号。系统自动通过短信给客户发送邮件信息。
	（有没有没有发票，先寄报告的。。）
12、	收款情况管理
超期三个月之内显示黄色，超期三个月以上显示红色，未超期显示绿色。
根据交款截止日期来判断，未到交款截止日期的，显示绿色，超过交款截止日期未满3个月的，显示黄色，超过交款截止日期3个月的，显示红色。
13、	业务逻辑审核
对不符合管理规范的委托单进行报警，比如提前邮寄发票和报告正本；折扣审批不符合规则；费用应收和实收不一致；缺材料；
14、	客户管理
维护长期客户基本信息，包括账期和折扣，上传合同扫描件。

二、	紫金的委托单
紫金的委托单是“特殊要求描述”字段里含有“紫金”。
生成账单的时候，让用户输入折后应收总金额和实收总金额，同时可以上传pdf文件。
    紫金的账单不用领导审批，系统自动审批通过，进入核销环节。
三、	日本样的委托单
日本样的委托单计算方法和国环的类似，也是企业—套餐的形式来计算费用。 具体企业-套餐标准另行提供。
注意：委托单自定义编号字段如果含有“Quality Control Sample” ,此样品不收费。
四、	市场局的委托单
市场局的委托单以ff开头。
通过导入excel文件（excel文件里主要字段是样品编号）来生成账单，生成账单的时候，让用户输入缴款单位、实收金额，并上传pdf文件。
市场局的账单不用领导审批，系统自动审批通过，进入核销环节。
五、	普通委托单
生成账单的时候，可以统一输入折扣，也可以让用户修改单个样品的折扣。
